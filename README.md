# Y2Z Travel

## task for Full Stack

### Setup Instruction

1. Clone Repository

```sh
git clone https://github.com/Belalirshad/AB-Movies-Backend.git
```

2. Install Dependencies

```sh
npm install
```

3. Running the Server

```sh
npm run dev
```

### Tech Stack Used

- React + Vite

- Tailwind CSS

- React Icon

- DND-Kit
